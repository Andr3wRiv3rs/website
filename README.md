# Devcord API
Backend repository available at https://github.com/devcord/api

## prerequisites
- node.js 12+

## tech stack
- vue
- webpack
- pug
- scss

## installation
```
git clone https://github.com/devcord/website devcord-website
cd devcord-website
npm install
```

## commands
```js
{
  "dev" // starts the development server
  "build" // builds the frontend
  "lint" // lint and fix
}
